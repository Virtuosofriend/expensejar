ExpenseJAR is a mobile application to trackdown the house expenses for couples, ideal for sharing household expenses, bills or even travel costs. The two of you can see exactly how much each is spending on what, as well as how those expenses change over time and fit in with the rest of your financial picture. 
No need to sync with your bank account or provide any account info, simply add a new expense, input date, cost and category and you're done!
Users can compare their expenses with previous months and with each other in order to balance any amounts owed.

## Features

- Shared homes for each user (up to two users per home)
- Add expenses per home
- Analytical history per home for all expenses per month of each year
- Comparison chart per month for each home
- Resolve month's expenses per home

## Tech

ExpenseJAR is build using VueJS 2.x, with the support of a Firebase database. UI is created using Vuetify 2.x and the mighty CapacitorJS for the APK creation.

## Installation

ExpenseJAR requires [Node.js](https://nodejs.org/) v14+ to run.

Install the dependencies and devDependencies and start the server.

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
